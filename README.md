## Local Development

This is a single-file static site — no build step required.

```bash
# Clone the repo
git clone https://github.com/voxility2010/<your-repo-name>.git

# Open in browser
open index.html
```

Or use a local server for best results:

```bash
# Using Python
python -m http.server 3000

# Using Node (npx)
npx serve .
```
